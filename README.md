# Express Blog Project (incomplete)
---

Simple blog application with authentication.

##   Setting Up
---

Change path of mongopath at ``config.json`` with your mongodb path.
```javascript
{
    "mongopath": "your-path"
}
```

```sh
$ npm install
$ npm start
```
